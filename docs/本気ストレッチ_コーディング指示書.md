## 6. GTMデータ属性

**⚠ GTMの設定・タグ埋め込みはVONDS側で行います。コーダーにはdata属性の付与のみ。dataLayer.pushの記述は不要。**

### CTAボタンに付与するdata属性

| 属性名 | 値 | 対象 |
|--------|-----|------|
| `data-gtm-click` | `cta-fv` | FVのCTAボタン |
| `data-gtm-click` | `cta-price` | 価格セクションのCTA |
| `data-gtm-click` | `cta-guarantee` | 返金保証後のCTA |
| `data-gtm-click` | `cta-final` | 最終CTAボタン |
| `data-gtm-click` | `cta-fixed-sp` | SP固定バナー |

### 電話リンクに付与するdata属性

| 属性名 | 値 | 対象 |
|--------|-----|------|
| `data-gtm-tel` | `header` | ヘッダー内電話番号 |
| `data-gtm-tel` | `footer` | フッター内電話番号 |
| `data-gtm-tel` | `fixed-sp` | SP固定バナーの電話 |

### 実装例

```html
<a href="tel:05088848993" data-gtm-click="cta-fv" data-gtm-tel="fv">精密姿勢診断を予約する</a>
```
