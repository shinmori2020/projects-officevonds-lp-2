Figma SP版デザイン（18枚）より抽出。2026/3/20作成。

---

## カラーパレット

```css
:root {
  /* メインカラー */
  --color-primary: #EC77AB;        /* ピンク — CTA・見出し強調・特典バッジ */
  --color-primary-light: #F8E0EF;  /* ピンク薄 — 比較アンカー背景・利用者変化セクション背景 */
  --color-dark: #1A1A1A;           /* ダーク — フッター背景・本文 */

  /* テキスト */
  --color-text: #1A1A1A;           /* 本文テキスト */
  --color-text-light: #666;        /* 補足・注釈テキスト */
  --color-text-white: #fff;        /* 白抜きテキスト（FV・ダーク背景上） */

  /* 背景 */
  --color-bg: #fff;                /* 基本背景 */
  --color-bg-pink: #F8E0EF;       /* ピンク薄背景セクション */
  --color-bg-dark: #1A1A1A;       /* ダーク背景（フッターコピーライト帯） */
  --color-bg-gray: #f5f5f5;       /* グレー背景（FAQ等） */

  /* アクセント */
  --color-accent-purple: #7B5EA7; /* 紫系 — 症状チェックのイラスト線・稲妻アイコン */

  /* ボーダー */
  --color-border: #ddd;            /* 一般的な区切り線 */
  --color-border-pink: #F0B0D0;   /* ピンク系の点線ボーダー（料金表・店舗情報） */
}
```

## タイポグラフィ

- **フォント**: Noto Sans JP（Google Fonts）
- **ウェイト**: 400（本文）/ 700（見出し・強調）/ 900（メインコピー特大）

### サイズ体系（SP基準）

| 用途 | サイズ目安 | 備考 |
|------|----------|------|
| メインコピー（FV） | 32〜40px相当 | 「精密姿勢診断」「深層筋ストレッチ」極太 |

| 本文 | 15〜16px | 説明文・FAQ回答 |
| 注釈・補足 | 13〜14px | ※注意書き・小文字 |
| CTAボタン | 18〜20px | 太字 |

## コンポーネントパターン

### CTAボタン
- 背景: `--color-primary`（#E91E8C）
- テキスト: 白、太字、18〜20px
- 角丸: 8px程度
- 幅: ほぼ全幅（左右padding付き）
- ボタン下に「毎月限定12名様」等の補足テキスト（黒・太字・中央寄せ）

### 特典バッジ（丸）
- ピンク背景の円形
- 白文字「特典 その1」等
- 右側にテキスト配置（横並び）

### 症状チェックイラスト
- 紫系の線画イラスト（男性シルエット）
- 稲妻マーク（痛みの表現）
- テキストと左右交互レイアウト
- キーワード（「重さ」「固まる」「動きづらい」「戻る」）はピンク色

### 料金テーブル
- 3列: コース / 通常料金 / 初回特典
- 点線ボーダー区切り（ピンク系点線）
- 初回特典の金額はピンク色・大きめ

### 3つのアプローチカード
- 白背景カード、ピンクのボーダー（左辺 or 上辺）
- 大きなピンク数字（1, 2, 3）
- 下にラベルテキスト
- 三角矢印で繋がるフロー表現

### FAQアイテム
- ピンク色の大きな「Q1」「Q2」等のナンバー
- 太字の質問文
- 通常ウェイトの回答文
- 点線ボーダーで区切り（アコーディオンではなく展開済み表示）

### 店舗情報テーブル
- ラベル（住所/電話/メール等）: ピンク色
- 値: ダーク色
- 点線ボーダー区切り
- 電話番号・メールはピンク色リンク

### フッター
- 「ご利用にあたって」セクション: 2列リンク配置
- SNSアイコン: LINE / Instagram / 電話（丸枠アイコン、横並び中央寄せ）
- コピーライト帯: ダーク背景、ピンク＋白文字

---

## 余白パターン

- セクション間: 48〜64px
- セクション内ブロック間: 24〜32px
- テキスト行間: 1.8〜2.0（日本語対応のゆったりめ）
- カード内padding: 16〜24px
- 左右padding（SP）: 20〜24px

---

## 装飾パターン

- **ピンク強調テキスト**: キーワードをピンク色で強調（多用）
- **点線ボーダー**: テーブル区切り・セクション区切りに使用（実線ではなく点線が基調）
- **＼ ／記号**: 「＼返金保証／」のような装飾見出し
- **丸バッジ**: 特典番号、連鎖アイコン（ピンク背景の丸）
- **写真背景 + 暗いオーバーレイ + 白テキスト**: FV・連鎖セクション・CTA②
- **ビフォーアフター写真**: 2枚並び＋角度数値のオーバーレイ
